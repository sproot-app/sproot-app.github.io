Sproot의 새로운 기능과 개선 소식입니다.

v1.1.0

곧 출시 예정

배움의 발자취가 보이는 가장 큰 업데이트. 생성은 더 확실하게, 조작은 더 자연스럽게.

배움의 발자취가 보여요

  • 학습량 그래프 — 일별 장수・학습 시간・정답률을 기간별로 돌아보기. 막대에 손을 대면 날짜별 기록이 보여요
  • 앞으로의 복습 — 잊어갈 타이밍에 맞춘 7일간의 복습 예보
  • 카드의 성장 — 미학습〜정착 분포를 한눈에
  • 복습의 원리를 그림으로 — 채점으로 다음에 나올 날이 정해지고, 간격이 1일→3일→8일→21일로 넓어져 '정착'하기까지를 앱 안에서 4단계로 설명해요(오늘의 복습 ⓘ에서)

새로운 기능

  • 만들 카드의 종류 선택 — 외국어 단어 / 용어・개념
  • 생성 중에도 다른 화면 사용 가능 — 진행 상황은 오른쪽 아래 작은 캡슐에서 확인. 앱을 닫거나 화면을 잠가도 돌아오면 이어서 자동으로 재개됩니다
  • 카드와 단어장 정렬 — 길게 눌러 손가락으로 원하는 순서로. 화면 끝까지 가져가면 자동으로 스크롤돼요
  • 오늘의 복습 접기 — 학습한 날에는 컴팩트하게 표시. ∨로 접어도 언제든 다시 펼 수 있어요
  • AI 모델 다운로드 백그라운드 지원 — 진행률은 Dynamic Island에 표시됩니다
  • 앱 표시 언어와 AI 카드 언어를 따로 설정
  • AI 채팅 음성 입력 지원 — 카드 상세에서 음성으로도 질문할 수 있어요
  • 작은 축하 연출 — 학습을 마치면 나무 주위로 잎이 흩날리고, 중요한 순간은 손끝으로도 전해집니다

개선

  • 사진 가져오기 강화 — 표・메모・단어만 있는 페이지를 자동으로 판별하고, 2단 리스트도 정확히 읽어요
  • 조작감을 새롭게 — 화면 가장자리 스와이프로 돌아가기, 시트를 잡아서 닫기, 단어장의 카드는 줄에서 부드럽게 부풀며 열립니다
  • 기기 내 AI를 고품질 모델로 일원화 — 옛 모델의 잔여 파일(약 3GB)은 첫 실행 시 자동 삭제되어 저장 공간이 돌아와요
  • 학습을 도중에 그만둬도 그때까지의 채점이 기록에 남아요
  • 예문 추가가 기존 예문과 겹치지 않게 됐어요
  • '동작 줄이기' 설정을 지원하고 버튼도 누르기 쉬워졌습니다
  • 앱 내 문구와 줄바꿈을 전면적으로 다듬었어요

버그 수정

  • 문장부호가 포함된 카드가 '다른 언어'로 잘못 판정되어 생성에 실패하던 문제를 수정했습니다
  • 요청한 장수보다 적게 만들어지거나 같은 카드가 섞이는 문제를 수정했습니다(30장을 부탁하면 정확히 30장)
  • 외국어 단어가 멋대로 번역되어 버리는 문제를 수정했습니다
  • AI 채팅이 인사에 뜻풀이로 답하는 경우가 있던 문제를 수정했습니다
  • 한국어 화면 일부에 일본어가 섞여 있던 문제를 수정했습니다

지원 환경: 이 버전부터 iOS 18 이상이 필요합니다.
